Nous proposons deux types de formation :


Une formation simple


Dans cette option on apprends a tous ceux qui le veulent les bases d'un domaine (dans la mecanique, dans l'informatique, dans le jardinage ou encore dans le bricolage), c'est un option qui durent entre 1 mois et 4 mois selon le choix du (ou des) domaine(s).Pour plus d'information que ce soit sur le prix de cette formation ou pour d'autre interrogation Contactez-nous!

Un stage


Vous êtes à la recherche d’une offre de stage ?

Nous proposons de possible stage avec nos prestataires dans les domaines de l'informatique, dans la mecanique, dans le jardinage ou encore dans le bricolage.
Nous sommes conscients que le stage est une étape importante dans le cursus d'un étudiant pour découvrir une entreprise, un secteur d’activité ou encore une profession, c'est pour cette raison qu'on propose cela au etudiant qui seront les travailleurs de demain.Le stage peut etre possiblement remunéré ( cela dépends de l'experience que vous possèdez et de la durée du stage ) .Pour plus d'information que ce soit sur le prix de cette formation ou pour d'autre interrogation Contactez-nous!
